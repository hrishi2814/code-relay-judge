Code Relay Problems

Problem 1: Hello Name

Write a program that reads a name from input and outputs "Hello, [name]!"

Input

A single line containing a name (string).

Output

A single line containing "Hello, [name]!"

Example

Input:
John

Output:
Hello, John!

Problem 2: Sum of Numbers

Write a program that reads a list of integers (one per line) and outputs their sum.

Input

Multiple lines, each containing a single integer.

Output

A single integer representing the sum of all input numbers.

Example

Input:
5
10
15
20

Output:
50

Problem 3: Palindrome Check

Write a program that reads a string and outputs "Yes" if it's a palindrome, "No" otherwise.

A palindrome is a string that reads the same backward as forward (e.g., "racecar").

Input

A single line containing a string.

Output

"Yes" if the input string is a palindrome, "No" otherwise.

Example

Input:
racecar

Output:
Yes

Input:
hello

Output:
No
